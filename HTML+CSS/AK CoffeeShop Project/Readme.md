# ☕ AK CoffeeShop Website

A modern and responsive Coffee Shop website built using **HTML5** and **CSS3**. This project was created as part of my HTML & CSS practice assignment to improve frontend web development skills.

---

## 📖 About the Project

AK CoffeeShop is a simple coffee shop landing page that showcases the shop's services, pricing plans, contact form, and footer information. The design uses a clean layout with a glassmorphism effect and smooth hover animations.

---

## 🚀 Features

- Responsive Navigation Bar
- Attractive Hero Section
- Coffee Services Section
- Pricing Cards
- Contact Form
- Modern Footer
- Hover Effects
- Glassmorphism Design
- Font Awesome Icons

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Font Awesome
