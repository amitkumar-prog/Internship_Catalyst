# 🎬 MovieHub - Movie Booking System

## 📌 Project Overview

MovieHub is a responsive Movie Booking System developed using HTML, CSS, and JavaScript. It allows users to browse movies, search and filter them, book tickets, view the booking summary, and confirm bookings. The project is designed with a modern dark theme and works on desktop, tablet, and mobile devices.

---

## 🚀 Features

- 🎥 Browse available movies
- 🔍 Search movies by name
- 🎭 Filter movies by category
- 🎟 Book movie tickets
- 💺 Select seat type
- 🧾 Automatic booking summary
- 💰 GST and Discount Calculation
- ⏳ Booking Countdown Timer
- 🌙 Dark Theme Toggle
- 🔄 Reset Booking Form
- 📱 Fully Responsive Design

## ⚙️ Functionalities

### Movie Section
- Display movie cards
- Search movies
- Filter by category

### Booking Section
- Customer details form
- Movie selection
- Ticket quantity
- Seat type selection

### Booking Summary
- Movie Name
- Seat Type
- Number of Tickets
- Subtotal
- GST (18%)
- Discount (10% on eligible bookings)
- Grand Total

### Timer
- 2-minute booking countdown
- Shows expiry message when time is over
