# 🛒 FreshCart – Smart Grocery Billing System

FreshCart is a responsive grocery shopping and billing web application built using **HTML, CSS, and JavaScript**. It allows users to browse products, add them to a shopping cart, enter customer details, and generate a bill with GST and discount calculations.

---

## 📌 Project Overview

The main objective of this project is to simulate a modern grocery shopping experience while practicing front-end web development concepts.

Users can:
- Browse featured grocery products
- Search and filter products
- Add or remove products from the cart
- Update product quantity
- Enter customer information
- Calculate GST and discount
- Generate the final bill
- View a limited-time offer with a countdown timer
- Switch between Light and Dark Mode
- Access the website on desktop, tablet, and mobile devices
