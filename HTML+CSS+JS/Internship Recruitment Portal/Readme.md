# 🎓 InternHire - Internship Recruitment Portal

A responsive **Internship Recruitment Portal** developed using **HTML, CSS and JavaScript**.

The project allows users to register internship candidates, manage candidate records, search applicants, check eligibility, view application summaries and experience a clean modern recruitment dashboard.

---

## 📌 Features

- Responsive Navigation Bar
- Modern Hero Section
- Candidate Registration Form
- Candidate Eligibility Check
- Add Candidate
- Delete Candidate
- Live Candidate Search
- Application Summary
- Interview Countdown Timer
- Theme Toggle (Dark / Light)
- Responsive Design
- Clean UI

---

## 🚀 Functionalities

### Candidate Registration

- Register internship candidates
- Store candidate information
- Validate required fields

### Candidate Management

- Display all registered candidates
- Search candidates by name
- Delete candidate records

### Eligibility Check

Candidates are automatically marked as:

- Eligible
- Not Eligible

based on age criteria.

### Application Summary

Displays

- Candidate Name
- Qualification
- Eligibility Status
- Interview Date

### Interview Timer

- Countdown Timer
- Auto notification when registration time expires

### Theme Toggle

- Dark Theme
- Light Theme
